Bus Ticketing System

This is a simple bus ticket booking web app built with **PHP**, **MySQL**, **AJAX** and **CSS/Bootstrap**.  
Users can search routes, select travel dates and book seats on regular bus lines.  
The project also includes a basic admin area for managing routes, schedules and bookings.
